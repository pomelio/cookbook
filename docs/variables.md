## variables

### string
```
'abc'
'i love you!'

```


### number
```
123
1.23
```


### boolean variables
```
true
false
```


### array
```
['abc', 'cde']
[1, 2, ...a]
[1, 3, 5, 6]
[1, 2, 3, 'abc', [1, 2, 3]]
```

> `...a` is an array variable spread

### Map
```
{ a: 1, b: 3, c: '123'}
{ a: [1, 2, 3], b: 'add'}
{ a: 1, b: { c: '122', d: 1}}
{ a: '3', ...abc}
```



> `...abc` is a map variable spread
